# VitalMetric

**Know your numbers. Own your health.**

A branded collection of health and fitness calculators targeting the 45+ demographic. Built with vanilla HTML/CSS/JS, hosted on Cloudflare Pages.

## Quick Start

Open `index.html` in a browser — no build step required.

## Structure

```
├── index.html              # Home / Metabolic Age Calculator
├── SPEC.md                 # Design & technology specification
├── css/
│   ├── design-tokens.css   # Colors, typography, spacing
│   ├── base.css            # Reset, typography, grid
│   └── responsive.css      # Breakpoints, layout
├── calculators/
│   └── metabolic-age.js    # Pure function calculator
├── assets/
│   ├── logo/               # Logo SVG + PNG
│   └── icons/              # Favicon, social icons
└── blog/                   # Blog articles
```
